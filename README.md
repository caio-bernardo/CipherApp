## Getting Started
 Make sure you have **npm** installed.
``` 
# Clone this repository
$ git clone https://github.com/caio-bernardo/CipherApp.git

# Install the dependecies
$ npm install

# Run all the tests
$ npm run test

# Start the application on your local enviroment
$ npm run dev
```
